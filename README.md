# CrowdChain — Decentralized Crowdfunding dApp
## BMIS2003 Blockchain Application Development

---

## Team Setup (Member 1 does this once, then shares repo)

### Prerequisites — install these first
- [Node.js v18+](https://nodejs.org)
- [MetaMask browser extension](https://metamask.io)
- [Ganache GUI](https://trufflesuite.com/ganache/) OR run `npx ganache`

---

## Step 1 — Clone & Install

```bash
git clone <your-repo-url>
cd crowdfunding-dapp

# Install Hardhat dependencies
npm install

# Install frontend dependencies
cd frontend
npm install
cd ..
```

---

## Step 2 — Start Ganache

Open **Ganache GUI** and click **Quickstart Ethereum**.

Make sure it's running on:
- RPC URL: `http://127.0.0.1:7545`
- Chain ID: `1337`
- Network ID: `5777`

---

## Step 3 — Connect MetaMask to Ganache

1. Open MetaMask → Add Network → Add manually:
   - Network name: `Ganache`
   - RPC URL: `http://127.0.0.1:7545`
   - Chain ID: `1337`
   - Currency symbol: `ETH`
2. Import a Ganache account: copy a private key from Ganache GUI → MetaMask → Import Account

---

## Step 4 — Compile & Deploy Contracts

```bash
# Compile contracts
npx hardhat compile

# Deploy to Ganache
npx hardhat run scripts/deploy.js --network ganache
```

This will:
- Deploy UserRegistry, RewardToken, and Crowdfunding contracts
- Transfer RewardToken ownership to Crowdfunding
- Auto-save contract addresses to `frontend/src/utils/addresses.json`

---

## Step 5 — Copy ABIs to Frontend

After compiling, copy ABIs:

```bash
mkdir -p frontend/src/utils/abi
cp artifacts/contracts/Crowdfunding.sol/Crowdfunding.json frontend/src/utils/abi/
cp artifacts/contracts/RewardToken.sol/RewardToken.json frontend/src/utils/abi/
cp artifacts/contracts/UserRegistry.sol/UserRegistry.json frontend/src/utils/abi/
```

---

## Step 6 — Run Frontend

```bash
cd frontend
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Common Issues

**MetaMask not connecting?**
- Make sure Ganache is running before opening the app
- Check that MetaMask is on the Ganache network (Chain ID 1337)

**"Must be registered" error?**
- Go to the login page and register your wallet first

**Deployment fails?**
- Make sure Ganache is running
- Check `hardhat.config.js` — the URL must match Ganache's RPC

**ABI not found?**
- Run `npx hardhat compile` first, then copy ABIs from `artifacts/`
